---
title: Cannot Edit Documentation
description: Fix the issue when unable to edit documentation.
---

## Symptoms

- Edit button is disabled
- Cannot type in the editor

## Common Causes

- Missing permissions
- Logged out of your account

## Quick Fix

<Steps>
  <Step title="Check login status">
    ````
    Make sure you are logged in. Log out and sign back in if needed.
    ````
  </Step>
  <Step title="Verify permissions">
    ````
    Ensure you have edit rights for this documentation space. Ask the admin to grant access if necessary.
    ````
  </Step>
</Steps>
