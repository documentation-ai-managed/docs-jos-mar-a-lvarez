## Overview

Welcome to José María Álvarez's documentation space. You use this platform to organize, manage, and share all your project documentation in one centralized location. Create structured pages, embed code examples, and collaborate seamlessly without leaving your workflow.

This space supports Markdown and MDX for rich content, including interactive components like tabs, steps, and diagrams. Start by exploring key features, then follow the quick start guide to set up your first document.

## Quick Start Guide

Get up and running in minutes. Follow these steps to create your first documentation page.

<Steps>
  <Step title="Create a New Space" icon="plus">
    Log in and select "New Documentation Space". Name it after your project, like "MyApp Docs".
  </Step>
  <Step title="Add Your First Page" icon="file-plus">
    Click "New Page" and choose MDX format. Start with frontmatter for title and description.
  </Step>
  <Step title="Organize Content" icon="folder">
    Use H2 headings (##) for sections. Add components like `<Callout>` for highlights.
  </Step>
  <Step title="Publish and Share" icon="share-2">
    Save and generate the site. Share the public URL with your team.
  </Step>
</Steps>
