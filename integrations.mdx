## Webhook Setup for Notifications

Set up webhooks to receive instant notifications on events like page updates or new comments. Webhooks POST data to your endpoint at `https://your-webhook-url.com/webhook`.

### Prerequisites
Ensure your endpoint handles POST requests with JSON payloads.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to Settings > Integrations > Webhooks. Click "New Webhook".
  </Step>
  <Step title="Configure Endpoint">
    Enter your URL and select events (e.g., `page.updated`, `comment.created`).

    <ParamField header="X-JMA-Signature" param-type="string" required="true">
      HMAC SHA-256 signature for payload verification.
    </ParamField>
  </Step>
  <Step title="Test and Activate" icon="check-circle">
    Send a test payload and activate the webhook.
  </Step>
</Steps>

Verify incoming payloads with this example:

<CodeGroup tabs="Node.js,Python">
  ```javascript
  const crypto = require('crypto');
  const signature = req.headers['x-jma-signature'];
  const expected = crypto.createHmac('sha256', 'YOUR_SECRET')
    .update(JSON.stringify(req.body))
    .digest('hex');
  if (signature !== expected) {
    return res.status(401).send('Invalid signature');
  }
  ```
  ```python
  import hmac
  import hashlib
  import json
  from flask import request

  payload = request.data
  signature = hmac.new(
    b'YOUR_SECRET',
    payload,
    hashlib.sha256
  ).hexdigest()
  if signature != request.headers.get('X-JMA-Signature'):
      return 'Invalid signature', 401
  ```
</CodeGroup>

## Embedding External Content

Embed videos, forms, or dashboards from external services into your docs.

<Tabs>
  <Tab title="Iframe Embed" icon="layout">
    Use secure iframes for simple embedding.

    ```
    <iframe
      src="https://example.com/dashboard"
      width="100%"
      height="500"
      frameborder="0">
    </iframe>
    ```

    <Callout kind="alert">
      Always use HTTPS sources to avoid mixed content warnings.
    </Callout>
  </Tab>
  <Tab title="API-Driven Embed" icon="code">
    Fetch and render dynamic content via API.

    ```javascript
    async function embedContent() {
      const response = await fetch('https://api.example.com/content', {
        headers: { Authorization: 'Bearer YOUR_TOKEN' }
      });
      const data = await response.json();
      document.getElementById('embed').innerHTML = data.html;
    }
    ```
  </Tab>
</Tabs>
