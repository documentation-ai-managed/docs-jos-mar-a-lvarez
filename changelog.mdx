## Upgrading Your Instance

Follow these steps to upgrade to the latest version safely.

<Steps>
  <Step title="Backup" icon="database">

    Create a full backup of your documentation data.

````bash
docs-cli backup --project your-project-id --output ./backup-$(date +%Y%m%d)
````

  </Step>
  <Step title="Update CLI" icon="download">

    Install the latest CLI version.

    <CodeGroup tabs="npm,yarn">
````bash
npm install -g @jose-maria-alvarez/docs-cli@latest
````
````bash
yarn global add @jose-maria-alvarez/docs-cli@latest
````
    </CodeGroup>

  </Step>
  <Step title="Deploy Update" icon="rocket">

    Run the upgrade command.

````bash
docs-cli upgrade --version 1.2.0 --api-key YOUR_API_KEY
````

  </Step>
</Steps>

<Callout kind="tip">
  Test upgrades in a staging environment first. Check the [migration guide](/migration) for version-specific instructions.
</Callout>
